# Accessibility Statement

## Our Commitment

Job-Magician is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

## Conformance Status

Job-Magician aims to conform with **Web Content Accessibility Guidelines (WCAG) 2.1 Level AA**. We strive to exceed these standards where possible.

### Current Conformance Level

- **WCAG 2.1 Level AA**: Substantial conformance (actively working toward full conformance)
- **Section 508**: Compliant
- **ADA Title II & III**: Compliant

## Accessibility Features

### Visual Accessibility

#### Screen Reader Support
- Full keyboard navigation support
- Semantic HTML structure
- ARIA labels and landmarks
- Skip navigation links
- Descriptive link text
- Alternative text for all images and icons
- Form labels and error messages

#### Visual Design
- High contrast color schemes
- Minimum 4.5:1 contrast ratio for normal text
- Minimum 3:1 contrast ratio for large text and UI components
- Resizable text up to 200% without loss of functionality
- No information conveyed by color alone
- Dark mode support

#### Visual Impairment Support
- Screen magnification compatible
- Focus indicators clearly visible
- No reliance on sensory characteristics alone (shape, size, visual location, orientation, sound)

### Motor/Physical Accessibility

#### Keyboard Navigation
- All functionality available via keyboard
- Logical tab order throughout the interface
- Visible focus indicators
- No keyboard traps
- Bypass blocks for repeated content

#### Input Methods
- Large clickable areas (minimum 44x44 pixels for touch targets)
- Adequate spacing between interactive elements
- Support for voice control software
- No time limits (or adjustable time limits with warnings)
- Error prevention and recovery mechanisms

### Cognitive Accessibility

#### Clear Communication
- Simple, plain language
- Consistent navigation and layout
- Predictable behavior
- Clear instructions and error messages
- Visual cues and multiple ways to navigate

#### Error Handling
- Clear error identification
- Suggestions for error correction
- Confirmation for important actions
- Ability to undo/redo actions
- Session timeout warnings with extension options

### Auditory Accessibility

#### Captions and Transcripts
- Captions for all video content
- Transcripts for audio content
- Visual alternatives for audio alerts
- No auto-playing audio

#### Sign Language
- **Deaf-First Design**: Built with visual communication as primary
- ASL video options where applicable
- Visual indicators for all audio cues
- Integration with VR4Deaf sign language tools

## Known Limitations

We are actively working to address the following known accessibility issues:

1. **Third-Party Content**: Some embedded third-party widgets may not fully meet accessibility standards
2. **PDF Documents**: Legacy documents are being converted to accessible formats
3. **Complex Data Visualizations**: Alternative text and data tables being enhanced

## Continuous Improvement

### Roadmap
We are committed to continuously improving accessibility:

#### Short-term (0-3 months)
- Complete automated accessibility scan remediation
- Enhance keyboard navigation
- Improve color contrast in remaining areas
- Add more descriptive ARIA labels

#### Medium-term (3-6 months)
- Implement comprehensive skip navigation
- Enhance error message clarity
- Improve form accessibility
- Add more sign language video content

#### Long-term (6-12 months)
- Full WCAG 2.1 Level AAA compliance (where feasible)
- AI-powered personalization for different disabilities
- Enhanced integration with VR4Deaf accessibility tools
- Multilingual accessibility support

### Monitoring
- Monthly accessibility metrics dashboard
- Quarterly user experience research with disabled users
- Annual third-party accessibility audit
- Continuous user feedback integration

## Legal Compliance

Job-Magician is designed to comply with:

- **ADA Title II**: State and local government services (applicable to TWC partnership)
- **ADA Title III**: Public accommodations and commercial facilities
- **Section 508**: Federal agency electronic and information technology
- **Section 504**: Rehabilitation Act (federally funded programs)

## Formal Complaints

If you are not satisfied with our response to your accessibility feedback:

### Internal Process
1. Request escalation to Accessibility Officer
2. Formal review within 10 business days
3. Written response with action plan

### External Process
You also have the right to file a complaint with:
- **U.S. Department of Justice** (ADA violations): [ADA.gov](https://www.ada.gov)
- **U.S. Department of Health and Human Services** (Section 504 violations)
- **Texas Workforce Commission** (TWC program accessibility)

## Third-Party Content

Job-Magician integrates with Texas Workforce Solutions and other third-party services. We:
- Require accessibility commitments from vendors
- Monitor third-party accessibility
- Provide alternative means to access third-party content when barriers exist
- Advocate for improved accessibility with partners

## Deaf-First Philosophy

As part of the 360 Magicians and VR4Deaf ecosystem, Job-Magician embodies a Deaf-First approach:

- **Visual-First Design**: All information accessible visually
- **No Reliance on Audio**: Audio is supplementary, never primary
- **Sign Language Integration**: ASL and other sign languages supported
- **Deaf Culture Awareness**: Built by and for the Deaf community
- **Universal Design**: Benefits all users, not just accommodations
